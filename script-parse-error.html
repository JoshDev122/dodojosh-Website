⚠️ Fix: Script Parse Error

This occurs when your GSC script has invalid syntax.

Solution

  • Check the compile log for the exact line causing the error.
  • Ensure all lines end with ; where required.
  • Make sure all { } brackets are matched correctly.
  • Check for missing function definitions or mismatched parameters.
